TFVENV User Documentation

Overview

tfvenv is a command-line application designed to manage virtual environments for Terraform and Terragrunt. It simplifies the setup, maintenance, and activation of isolated environments, ensuring consistent tooling versions and configurations across different projects and teams.

Commands

create

Description: Creates a new virtual environment with specified Terraform and Terragrunt versions.

tfvenv create --env=<env_directory> --tf-version=<version> --tg-version=<version> --environments=<env1,env2,...>

delete

Description: Deletes an existing virtual environment.

tfvenv delete --env=<env_directory>

list

Description: Lists all environments managed by tfvenv.

tfvenv list --env=<env_directory>

Environment Variables

TFVENV_LOG_FILE: Specifies the file path where tfvenv will write its logs.

export TFVENV_LOG_FILE=/path/to/tfvenv.log

TFVENV_VERBOSE: Enables verbose (debug-level) logging when set to true.

export TFVENV_VERBOSE=true

Examples

Create multiple environments:

tfvenv create --env=.tfvenv --tf-version=1.3.5 --tg-version=0.35.12 --environments=dev,prod

Activate an environment with custom variables:

tfvenv activate --env=.tfvenv --env-type=staging --var=API_KEY=abcdef,DEBUG=true
