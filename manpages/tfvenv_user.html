TFVENV Developer Documentation

Installation

Prerequisites:

  • Go: Download from https://golang.org/dl/
  • Git: Download from https://git-scm.com/downloads

Steps to Build the Application

  1. Clone the repository:
    2. git clone https://github.com/rickcollette/tfvenv.git
  2. Navigate to the project directory:
    3. cd tfvenv
  3. Build the application:
    4. go build -o tfvenv main.go
  4. Move the binary to a directory in your PATH:
    5. mv tfvenv /usr/local/bin/
  5. Verify the installation:
    6. tfvenv --help

Contributing

To contribute to tfvenv, follow these steps:

  1. Fork the repository:
    2. git clone https://github.com/yourusername/tfvenv.git
  2. Create a new branch:
    3. git checkout -b feature/your-feature-name
  3. Make your changes.
  4. Commit your changes:
    5. git commit -m "Add feature: your-feature-name"
  5. Push to your fork:
    6. git push origin feature/your-feature-name
  6. Open a pull request.
